## alpacon login

Log in to Alpacon

### Synopsis

Log in to Alpacon. To access Alpacon, workspace url is must specified

```
alpacon login [flags]
```

### Examples

```

	alpacon login

	alpacon login [WORKSPACE_URL] -u [USERNAME] -p [PASSWORD]
	alpacon login example.alpacon.io 
	
	# Login via API Token
	alpacon login -w [WORKSPACE_URL] -t [TOKEN_KEY]
	
```
